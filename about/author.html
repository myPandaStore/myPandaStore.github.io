    # 介绍

    我是宜觉，热爱生活，享受思考，乐于前端给予我的正反馈。

    目前研一(2022),不知不觉间，迈入一个新的大学校园也将近一年，从对研究生活的无限憧憬到对研究生活的失望只是一瞬之间。时常惶恐短短几十载，莫不如是弹指一挥间。往事种种，皆如云烟。索性换了赛道，至此半年，也算有了不少的收获。常言到"星光不负赶路人"，愿每一次回首，都有值得庆幸的瞬间。

    // 更新 不知不觉又过一年，现在已然是来到秋招的日子，必然会经历的彷徨，选择，总是常伴左右。能够一步步的确认自己的成长其实已经是很棒的一段又一段光景。

    # 我的经历

    目前研一(2022)准备实习，就读于武汉理工大学。非科班转码，在新的人生阶段又有了自己些许的感悟。

    稻盛和夫曾说“你缺乏的也许不是一份热爱的工作，而是热爱一件事的能力”。对此，我有着自己的一些看法。我本科就读于传统工科，研究生专业也是传统工科，没有选择跨考计算机专业，大概是没有勇气舍弃自己曾经学过的东西吧，好像这样就算否定了自己的前四年一样。但，有舍才有得。当我认清自己对于传统工科的热情缺乏和学校的学术氛围后，转码成为了我的一次转折。我开始全身心的投入前端，贪婪的汲取自己所想要的知识。当我们基于自己的兴趣去学习，往往能够事半功倍。

    "行百里者半九十"，我们应该要对自己做的每一件事做好充足的准备，并坚持完成自己的计划。在经历了两次失败的大厂实习面试后，我感受到自己的不足。随之继续巩固所学基础，和练习算法，锻炼自己的coding能力。

    // 更新 一程有一程的风景，一程有一程的陪伴，坚定的做自己喜欢的东西应该就算式不错的生活方式了。
